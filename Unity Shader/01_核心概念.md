# Unity Shader - 第 1 课：从 GLSL 思维切到 Unity Shader

## 学习目标

本节结束后，你应该能做到：

- 说清 Unity shader 文件里 ShaderLab、Properties、SubShader、Pass、CGPROGRAM 各自负责什么。
- 把 Unity 的 `Graphics.Blit` 后处理流程对应到 OpenGL 的 fullscreen quad pass。
- 理解 C# 的 `Material.SetTexture`、`Material.SetFloat` 如何对应 shader 里的 texture 和 uniform。
- 看懂 `CameraBloodEffect.shader` 的整体结构，不要求逐行完全掌握混合公式。

## 内容讲解

### 1. Unity Shader 不是纯 HLSL

在 OpenGL/GLSL 里，你通常直接面对这些东西：

```text
shader program
uniform
sampler2D
vertex shader
fragment shader
framebuffer
draw fullscreen quad
```

Unity 把它们包在一个更工程化的系统里：

```text
ShaderLab     Unity shader 文件的外壳，负责声明名字、属性、Pass、渲染状态
Properties    暴露给 Material 和 Inspector 的参数入口
SubShader     一套可运行的渲染方案
Pass          一次实际 draw call 的渲染通道
CGPROGRAM     真正写 vertex/fragment shader 的地方
Material      shader + 参数值的容器
```

### 2. 后处理就是 fullscreen quad

教程里的血屏效果是后处理。OpenGL 思维下，它等价于：

```text
1. 相机先把场景渲染到一张 texture
2. 绑定目标 framebuffer
3. 画一个 fullscreen quad
4. fragment shader 采样场景 texture
5. 再采样血迹 texture
6. 输出混合后的像素
```

Unity 的写法是：

```csharp
void OnRenderImage(RenderTexture src, RenderTexture dest)
{
    Graphics.Blit(src, dest, material);
}
```

`Graphics.Blit` 替你完成 fullscreen quad 绘制。`src` 会自动传给 shader 里的 `_MainTex`。

### 3. 参数怎么传进 shader

C# 里：

```csharp
material.SetTexture("_BloodTex", bloodTexture);
material.SetTexture("_BloodBump", bloodNormalMap);
material.SetFloat("_BloodAmount", bloodAmount);
material.SetFloat("_Distortion", distortion);
```

Shader 里：

```hlsl
sampler2D _BloodTex;
sampler2D _BloodBump;
float _BloodAmount;
float _Distortion;
```

名字必须一致。Unity 靠字符串名把 C# 传入的数据绑定到 shader 变量。

### 4. `Properties` 和 HLSL 变量不是一回事

`Properties` 是给 Unity 编辑器和 Material 系统看的：

```hlsl
Properties
{
    _BloodTex("Blood Texture", 2D) = "white"{}
}
```

真正给 fragment shader 使用，还要在 `CGPROGRAM` 里声明：

```hlsl
sampler2D _BloodTex;
```

只写 `Properties`，shader 代码里不能直接用；只写 `sampler2D`，Inspector 里不会自动出现可拖拽槽位。二者经常同名配套使用。

## 小结

1. Unity Shader = ShaderLab 外壳 + HLSL/CG shader 代码。
2. `Graphics.Blit` 就是 Unity 封装的 fullscreen quad 后处理。
3. 后处理里的 `_MainTex` 通常是相机渲染好的屏幕图。
4. `Material.SetTexture/SetFloat` 相当于给 shader 设置 uniform/sampler。
5. `Properties` 负责编辑器暴露，`CGPROGRAM` 里的变量负责 shader 实际使用。
