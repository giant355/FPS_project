# English Vibe Coding Vocabulary Log

- **convention** - 约定俗成的做法；大家通常遵守的命名规则或编码习惯
  - Example: `_MainTex` is a Unity convention for the main texture.

- **overload** - 重载；同名函数或方法因为参数不同而形成的不同版本
  - Example: `Graphics.Blit` has several overloads.

- **nuance** - 细微但重要的差别，通常指意思、行为或用法上的微妙区别
  - Example: The nuance is that `_MainTex` is a convention, not a hard rule.

- **semantic** - 语义；在 shader 里指告诉渲染管线某个变量代表什么的标签
  - Example: `SV_Target` is a fragment output semantic.

- **clean** - 清晰、简洁、容易理解；常用来形容代码、解释或设计不杂乱
  - Example: This is a clean explanation of the rendering flow.

- **arbitrary** - 任意的；没有固定规则或明确理由，取决于选择者自己决定
  - Example: The variable name is arbitrary, but the semantic is not.

- **internally** - 在系统内部；由底层实现或框架内部处理，而不是你直接写出来的逻辑
  - Example: Unity handles the source texture internally during `Graphics.Blit`.

- **accurate** - 准确的；表达和事实一致，并且足够精确
  - Example: That mental model is accurate for this post-processing pass.
